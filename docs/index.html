NeurINO

2D-to-3D Knowledge Transfer for Neuron Segmentation

Abstract

NeurINO is a framework that transfers powerful 2D self-supervised representations from DINOv3 to the 3D neuron segmentation domain. Built on MedNeXt, NeurINO outperforms existing methods on multiple neuron datasets.
